# Hemanth Kumar K - Portfolio

A highly premium, interactive, and beautifully designed personal portfolio website showcasing my expertise in Embedded Systems, IoT, and Digital Experiences. 

Built with modern web technologies and designed with a strict focus on minimal, iPhone-style glassmorphism, fluid animations, and a dynamic tracking glow effect.

## 🌟 Live Website
**[https://hemanthnanu-tech.github.io/Hemanth-Kumar-K-protfolio/](https://hemanthnanu-tech.github.io/Hemanth-Kumar-K-protfolio/)**

## 🛠 Technologies & Architecture
- **React 18**: Component-based UI architecture
- **Vite**: Blazing fast build tool and development server
- **Tailwind CSS**: Utility-first styling with custom glassmorphism extensions
- **Framer Motion**: Smooth, physical animations and scroll reveals
- **Lucide Icons**: Clean, professional iconography

## ✨ Key Features
- **Premium Global Typography**: Fully integrated with the `Outfit` font for a high-end editorial feel.
- **Dynamic Glassmorphism**: High-end frosted glass panels (`backdrop-filter: blur(40px)`) with physical hover translations and soft tab-shadows.
- **Mouse Tracking Glow**: A soft, custom ambient light that perfectly tracks the user's cursor across the entire page, built entirely with optimized CSS variables.
- **Dual-Themed Support**: Seamless light and dark mode toggling.
- **Categorized Projects**: Distinct sections separating full-stack Software Apps from Academic/Hardware prototypes.
- **Interactive Certificates**: Official verification documents (PDF/JPEG) linked directly to the educational timeline.

## 🚀 Local Development
To run this project locally on your machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/hemanthnanu-tech/Hemanth-Kumar-K-protfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd Hemanth-Kumar-K-protfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

## 📦 Deployment
This project is fully automated and configured to deploy to **GitHub Pages** using GitHub Actions whenever a change is pushed to the `main` branch.
